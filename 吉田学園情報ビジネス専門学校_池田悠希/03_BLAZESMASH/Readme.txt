BLAZESMASH

3年時にC++でチームで制作したスポーツゲームです。
サブプログラマーとしてR&Dをメインに制作を行いました。
波動エフェクトを制作するためにステンシルとレンダリングテクスチャの勉強を新しくしたので見てほしいです。
制作人数
4人
制作期間
3.5ヶ月(300時間）

操作方法
～キーボード～
・P　        : ポーズ（試合中のみ）
・WASD       : アウトゲームではメニュー操作、インゲームでは移動、しゃがみ、急降下
・SPACE      : ジャンプ、二段ジャンプ
・ENTER      : 通常攻撃（スイング、チャージスイング、スマッシュ、スパイク、起き上がり吸収、必殺技）
・UP         : キャッチ
・RIGHT      : バント

～コントローラ～
・スタートボタン : ポーズ（試合中のみ）
・左スティック   : アウトゲームではメニュー操作、インゲームでは移動、しゃがみ、急降下
・Aボタン        : ジャンプ、二段ジャンプ
・Xボタン        : 通常攻撃（スイング、チャージスイング、スマッシュ、スパイク、起き上がり吸収、必殺技）
・Bボタン        : バント
・Yボタン        : キャッチ
